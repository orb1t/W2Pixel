wPointPath( );

Com isto você consegue colocar varios pixels únicos interligados entre si atráves de suas posições de uma vez. Sua propriedade é um valor em Json.

Para um exemplo, primeiro defina um ponto inicial, veja: 


 < !DOCTYPE HTML >
 < html >
 < head >

	< script src="W2Pixel-2.2.0.min.js" >< /script >

 < /head >
 < body >

	< script >

        w2pixel( 400 , 400 , 10 , 10 );
        wPointPath([ 2,2 ]);

	< /script >

 < /body >
 < /html >

Execultando no navegador:

A partir deste ponto inicial, poderá escolher outra posição, veja: 


 < !DOCTYPE HTML >
 < html >
 < head >
					   
	 < script src="W2Pixel-2.2.0.min.js" >< /script >
					   
 < /head >
 < body >
					   
	< script >
					   
        w2pixel( 200 , 200 , 10 , 10 );
        wPointPath([ 2,2,0,1 ]);
		
	< /script >
					   
 < /body >
 < /html >

Execultando:

Para o melhor entender do que foi feito no exemplo acima, a sintaxe é a seguinte:

wPointPath([ x,y,x,y ]);
O ponto inicial foi de dois pixels no eixo X e dois pixels no eixo Y, para gerar o segundo ponto, foi movimentado zero pixel no eixo X e mais um pixel no eixo Y.

Observe a continuação deste exemplo: 


< !DOCTYPE HTML >
< html >
< head >
                                          
    < script src="W2Pixel-2.2.0.min.js" >< /script >
                                          
< /head >
< body >
                                          
    < script >
                                          
        w2pixel( 200 , 200 , 10 , 10 );
        wPointPath([ 2,2,0,1,1,0,1,0,0,-1,3,3,0,1,0,2 ]);
                           
    < /script >
                                          
< /body >
< /html >

Execultando:

Sempre deve informar a posição X e Y de cada ponto que for gerar, e esse ponto sempre se posicionara a partir da posição do último pixel gerado.
