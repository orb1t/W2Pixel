wLinePath( );

Funciona como o wPointPath( );, só que com umas diferenças, pois wLinePath( ); gera linhas em vez de pontos e não se posiciona a partir da última posição, sua propriedade é um valor Json.

Vamos para um exemplo, para melhorar o entendimento desta função:

Primeiro definimos um ponto inicial: 


 < !DOCTYPE HTML >
 < html >
 < head >

	< script src="W2Pixel-2.2.0.min.js" >< /script >

 < /head >
 < body >

	< script >

        w2pixel( 400 , 400 , 10 , 10 );
        wLinePath([ 1,1 ]);

	< /script >

 < /body >
 < /html >

Execultando no navegador:

Agora vamos escolher outra posição, isso vai fazer voçê lembrar da tag line de SVG: 


 < !DOCTYPE HTML >
 < html >
 < head >
					   
	 < script src="W2Pixel-2.2.0.min.js" >< /script >
					   
 < /head >
 < body >
					   
	< script >
					   
        w2pixel( 200 , 200 , 10 , 10 );
        wLinePath([ 1,1,8,1 ]);
		
	< /script >
					   
 < /body >
 < /html >

Execultando:

Vamos escolher mais uma posição, agora vai fazer voçê lembrar da tag polyline de SVG: 


< !DOCTYPE HTML >

        wLinePath([ 1,1,8,1,8,8 ]);

< /html >
