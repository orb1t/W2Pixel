wCreateObject( ); e wObject( );

wCreateObject( ); é para ajudar na tarefa de criar objetos (formas) complexas e wObject( ); é para aplicar estes objetos (formas);

Primeiro sobre o wCreateObject();, ao chamar esta função você cria uma réplica de sua tela de desenho, mas sensível ao mouse, desta forma é possível criar objetos (formas) mais complexos com mais facilidade, está função possui três propriedades, que são a largura e altura da replica a ser gerada, e a última é um link (indereço web) ou caminho entre pastas de um arquivo de imagem.

A réplica de sua tela de desenho, será gerada ao lado da original, e também aparecerá um pequeno retângulo onde o Json do objeto que estará desenhando será gerado, veja: 


 < !DOCTYPE HTML >
 < html >
 < head >

	< script src="W2Pixel-2.3.0.min.js" >< /script >

 < /head >
 < body >

	< script >

        w2pixel( 200 , 200 , 10 , 10 );
        wFill( "#0af" );
        wRect( 4,4,2,2 );
        wCreateObject( 300,300 );

	< /script >

 < /body >
 < /html >

( pode interagir com o resultado a baixo )

Exitem dois controles de mouse, quando a função é chamada, se você clicar em um dos pixels da réplica, o pixel passa a ser preto e na tela de desenho original mostra como vai ficar, mas o controle esta em "CLICK", é possível mudar para um controle mais rápido, basta apenas pressionar a tecla "Q" do seu teclado e o controle passará a ser "ONMOUSEENTER" ( tente fazer isso no exemplo acima ), para voltar ao "CLICK" basta pressionar a mesma tecla novamente, a tecla "Q" ela muda o tipo de controle.

Se você errou, use o CTRL+Z para desfazer e se desejar recomeçar a criar o objeto, basta pressionar a tecla "W" (w) que um reflesh será dado na página ( tente no exemplo acima ), a tecla W recomeça a sua criação.

Depois de criar o seu objeto (forma), copie o código Json do pequeno retângulo, apenas click duas vezes sobre o código e pressione Ctrl+C de seu teclado e cole no wObject( ); que o seu objeto (forma) será aplicado na tela de desenho original. 


 < !DOCTYPE HTML >
 < html >
 < head >
					   
	 < script src="W2Pixel-2.3.0.min.js" >< /script >

Se você tiver uma imagem de algo que queira reproduzir utilizando o W2Pixel, você pode utilizar um indereço desta imagem e fazer exatamente isto: 


< !DOCTYPE HTML >
< html >
< head >
                                      
    < script src="W2Pixel-2.3.0.min.js" >< /script >
                                      
< /head >
< body >
                                      
    < script >
                                      
        w2pixel( 250 , 250 , 10 , 10 );
        wCreateObject( 300,300,"../../../img/w2ps.png");
                       
< /script >
                                      
< /body >
< /html >

A imagem é carregada no fundo do editor para facilitar seu trabalho, com esta funcionalidade será mais fácil passar sua arte para o W2Pixel.

Isto tudo serve como uma ferramenta para ajudar você nas sua criações com o W2Pixel, a realizar tarefas mais complexas.
